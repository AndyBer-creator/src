Описание программы:

    Интерфейс BankClient: Определяет методы для работы с банковским клиентом.
    Структура Client: Реализует интерфейс BankClient, содержит методы для депозита, снятия и получения баланса.
    Использует sync.Mutex для обеспечения безопасного доступа к балансу из горутин.
    Горутины:
        depositRoutine: Зачисляет случайную сумму на счет клиента каждые 0.5-1 секунды.
        withdrawalRoutine: Снимает случайную сумму с счета клиента каждые 0.5-1 секунды, 
        обрабатывая ошибки при недостатке средств.
    Основной цикл: Обрабатывает команды пользователя для проверки баланса, зачисления и снятия средств,
    а также завершения работы программы.
